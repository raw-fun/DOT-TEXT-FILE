৮.৩ মূল অটোমেশন লুপ - লাইন 635-760
────────────────────────────────────

5-পর্যায় চক্র (N iterations):

┌─────────────────────────────────┐
│ Iteration Counter: $i = 1 to N │
└─────────────────────────────────┘
       │
       ├─→【PHASE 1: CHECK】
       │   └─ Element availability verification
       │      ├─ Is Displayed?
       │      ├─ Is Enabled?
       │      └─ Is Interactable?
       │
       ├─→【PHASE 2: INPUT】
       │   └─ Text entry
       │      ├─ Clear textarea
       │      ├─ Type "NEXT"
       │      └─ Verify input
       │
       ├─→【PHASE 3: TRIGGER】
       │   └─ Action execution
       │      ├─ Locate send button
       │      ├─ Pre-click verification
       │      └─ Click (with JS fallback)
       │
       ├─→【PHASE 4: WAIT】
       │   └─ Response completion detection
       │      ├─ Method 1: Loading indicator
       │      ├─ Method 2: Text stability
       │      └─ Method 3: DOM Mutation Observer
       │
       └─→【PHASE 5: DELAY】
           └─ Anti-detection pause
              ├─ Random 3-7s (Gaussian)
              └─ Progress display

Loop:  if $i < N → Return to PHASE 1
      else → Proceed to SHUTDOWN

৮.৪ শাটডাউন সিকোয়েন্স - লাইন 762-788
─────────────────────────────────────

5-ধাপ পরিচ্ছন্নতা:
AUTOMATION COMPLETE
  ├─ [1] Final logging (stats, time, success rate)
  ├─ [2] Data persistence (responses, session)
  ├─ [3] Browser cleanup (cache, tabs)
  ├─ [4] WebDriver disposal (quit, release)
  └─ [5] User notification (success, logs, exit)

৮.৫ ব্যতিক্রম ব্যবস্থাপনা - লাইন 790-810
──────────────────────────────────────────

6টি প্রধান ত্রুটি প্রকার: 

[E1] Element Not Found
     Cause: UI changes, page load issues
     Recovery: Wait → Retry → Alternative selectors → Screenshot

[E2] Timeout Exception
     Cause: Slow network, long AI processing
     Recovery: Extend timeout → Check network → Pause

[E3] Session Lost
     Cause: Logout, cookie expiry
     Recovery: Detect → Pause → Notify → Resume

[E4] Network Disconnection
     Cause: Internet loss
     Recovery:  Detect → Wait 5min → Retry → Save state

[E5] Bot Detection / CAPTCHA
     Cause: Anti-automation
     Recovery: Pause → Alert → Wait manual solve → Resume

[E6] Browser Crash
     Cause: Memory leak, driver issue
     Recovery: Detect → Save → Restart → Resume

Prevention strategies included for each error type

workflow, automation loop, startup, shutdown, error handling, phases
